# 🚀 Quick Setup Guide

## Step 1: Get Your Remove.bg API Key (FREE)

### A. Go to Remove.bg
1. Open https://remove.bg/api in your browser
2. Click **"Sign up"** (top right)
3. Create a free account (email + password)
4. Confirm your email

### B. Get Your API Key
1. After login, go to your **API Dashboard**
2. Find your **API Key** (looks like: `abc123def456xyz...`)
3. Copy it to clipboard

## Step 2: Open the App

### Option A: Direct (Easiest)
- Double-click `index.html` from File Explorer
- Your browser opens automatically

### Option B: Via Terminal
```powershell
cd c:\Users\VRUSHTI\Desktop\edit
python -m http.server 8000
# Then open: http://localhost:8000
```

## Step 3: Paste Your API Key

1. **Paste** your Remove.bg API key into the text field at the top
   - Field says: "Paste your Remove.bg API key here"
2. Click **"Save"** button
3. You'll see: ✅ "API key saved successfully!"

## Step 4: Start Using!

1. **Upload** your kurti photo (click or drag & drop)
2. Click **"🚀 Remove Background"** 
3. Wait 3-5 seconds for processing
4. **Select** any of 10 background scenes
5. **Download** your result as PNG
